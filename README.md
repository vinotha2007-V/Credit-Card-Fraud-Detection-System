# 💳 Credit Card Fraud Detection System

A Machine Learning project that detects fraudulent credit card transactions using classification algorithms and imbalance handling techniques.

## 🚀 Features
- Fraud transaction detection
- SMOTE for imbalanced data handling
- XGBoost classification model
- Flask API deployment
- Streamlit dashboard support

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- XGBoost
- Flask
- Streamlit

## 📂 Project Workflow
1. Data Preprocessing  
2. Feature Scaling  
3. SMOTE Oversampling  
4. Model Training  
5. Fraud Prediction  
6. Deployment  

## ▶️ Run Project

```bash
pip install -r requirements.txt
python main.py
```

## 🌐 Run Flask API

```bash
cd app
python app.py
```

## 📊 Model
- XGBoost Classifier
- High Accuracy & Recall
- Optimized for fraud detection
